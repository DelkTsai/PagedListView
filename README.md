# PagedListView
Scroll ListView faster and more accurate

ListView等滚动位置经常不符合用户预期：很多时候都是看完一页想滑到下一页，但滑动一次距离往往不是不够就是超过。\n使用PageScoller可以在 中低速滑动 后会滚动一页，很慢滑动 或 很快滑动 则按默认处理。
